# Day 1 — Project Planning 📋

## Project Title
AI Resume Analyzer

## Problem Statement
Recruiters spend too much time manually reading resumes.
This project automates resume analysis using AI and NLP.

## Goal
Build an AI system that:
- Extracts skills from resumes automatically
- Scores resumes against job descriptions
- Recommends courses for missing skills
- Provides a web UI for easy use

## Tools & Tech Stack
| Tool | Purpose |
|------|---------|
| Python | Core programming language |
| spaCy | NLP and skill extraction |
| Scikit-learn | TF-IDF scoring |
| Pandas | Data processing |
| Matplotlib | Data visualization |
| Streamlit | Web app UI |
